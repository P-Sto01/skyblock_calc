# Skyblock-calc
### An open-sourced price calculator for Hypixel Skyblock.
This project's goal is to provide a free python aplication to calculate the most optimal way to craft an item.
## How to use:
Use the `getrecipes.py` file to generate a json file conataining all the recipes available on the server(using the [NEU-Repo](https://github.com/NotEnoughUpdates/NotEnoughUpdates-REPO)). The rest of the proccess is unknown for now ;)

### Generate GRAPH Vizualization
Run `.\visualize_recipes.py --item GHAST_CLOAK --depth 2 --max-nodes 80 --save .\ghast_cloak_graph.png`
